# Лабораторная работа 3: Анализ протокола HTTP

## Цели работы:
- Изучение основ работы протокола HTTP
- Овладение навыками работы с Wireshark
- Анализ HTTP-запросов и ответов
- Исследование механизмов кэширования

## Выполненные задания:

### Задание 1: Загрузка файла трассировки
- Анализ общего количества пакетов и объема данных
- Статистика HTTP-запросов и ответов

### Задание 2: Изучение трассировки запроса и ответа  
- Анализ GET-запроса и HTTP-заголовков
- Изучение структуры HTTP-ответа

### Задание 3: Кэширование контента
- Исследование механизма If-Modified-Since
- Анализ ответа 304 Not Modified

### Задание 4: Сложные страницы
- Статистика HTTP-пакетов
- Распределение нагрузки по серверам

## Технологии:
- Wireshark 4.4.0
- HTTP/1.1

## Результаты:
В ходе работы изучены принципы работы протокола HTTP, освоены методы анализа сетевого трафика, исследованы механизмы кэширования и оптимизации веб-страниц.
